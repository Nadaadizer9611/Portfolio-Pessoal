Visão Geral do Projeto

Este projeto tem como objetivo desenvolver um sistema especializado voltado para a informatização da gestão de barbearias, com foco na automação de processos internos e melhoria da experiência do cliente. A solução foi pensada para atender às necessidades específicas da Barbearia FJ, situada em São Lourenço do Sul, sendo construída com base em demandas reais do proprietário.

Objetivo do Sistema

Objetivo principal:

Criar uma aplicação capaz de gerenciar agendamentos, cadastro de clientes, controle de estoque e fluxo de caixa de forma integrada e automatizada.

Funcionalidades principais:

  • Agendamento de atendimentos com notificações e histórico.
  • Cadastro de clientes, funcionários, gerentes e fornecedores.
  • Controle de estoque com alertas de níveis baixos.
  • Registro de movimentações financeiras no caixa.
  • Automatização de processos entre atendimento, estoque e financeiro.

Arquitetura e Desenvolvimento

O sistema foi modelado utilizando diagramas fundamentais de engenharia de software:

  • Diagrama de Contexto – Visão geral das interações entre usuários e o sistema.
  • Diagrama de Fluxo de Dados (DFD) – Detalhamento do tráfego de informações.
  • Diagrama Entidade-Relacionamento (DER) – Representação do banco de dados relacional.

Tecnologias utilizadas:

  • Linguagem: Java
  • Banco de Dados: MySQL
  • Execução: Desktop
  • Interface: Intuitiva e limpa, com foco na usabilidade

Requisitos do Sistema

Requisitos funcionais:

  • Cadastro de clientes, funcionários, gerente e fornecedores
  • Agendamento, edição e exclusão de horários
  • Integração entre atendimento, estoque e caixa
  • Alertas de produtos em baixa quantidade

Requisitos não funcionais:

  • Execução fluida e responsiva
  • Segurança na manipulação de dados
  • Compatibilidade com sistemas desktop
  • Comunicação eficiente com banco de dados MySQL
